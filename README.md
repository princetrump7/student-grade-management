Student Grade Management System
CS 101 – C++ Programming Project

Program Description:
This program is a console-based Student Grade Management System written in C++.
It allows the user to enter student records, compute grades, display reports,
show class statistics, and search for students using their student ID.

How to Compile:

1. Open the program in Code::Blocks, Vs Code, or any C++ compiler.
2. Save the file as main.cpp.
3. Compile and run the program.

Using g++ compiler:
g++ main.cpp -o gradeSystem

How to Run:
After compiling, run the executable file and follow the menu instructions shown on screen.

Known Limitations:

* Maximum number of students is 30.
* Data is not stored permanently.
* The program runs only in console.

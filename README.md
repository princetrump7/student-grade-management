# Student Grade Management System

## Project Description

This project is a simple Student Grade Management System written in C++.  
It was developed as a CS 101 project to demonstrate the use of:

- Arrays
- Functions
- Loops
- Conditional statements
- Pointers
- Searching
- Input validation

The program allows the user to enter student records, calculate grades, display reports, and search for students by ID.

---

## Features

The system can:

1. Enter student records
2. Calculate total scores
3. Assign letter grades
4. Display full class report
5. Show class statistics
6. Search for a student by ID

---

## Grading System

| Score Range | Grade |
|-------------|-------|

| 70 - 79     | B     |

| 50 - 59     | D     |
